# NovaBuilder Documentation

This directory contains all **official documentation, audits, specifications and internal references**
for the NovaBuilder ecosystem.

It is intentionally structured to support:

- Long-term maintainability
- Evidence-based development
- Security, compliance & auditability
- Clear separation between developer-facing and release-facing artefacts
